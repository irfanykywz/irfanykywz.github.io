Halo teman-teman! 👋 Di sini kita akan belajar cara mengatur Python di Windows. Yuk, simak langkah-langkahnya!

## Langkah 1: Download Python

1. Kunjungi situs resmi Python di [python.org](https://www.python.org/downloads/).
2. Klik tombol "Download Python" yang sesuai dengan versi terbaru.
3. Setelah selesai, buka file installer yang sudah di-download.

## Langkah 2: Instal Python

1. Pada jendela installer, centang opsi **"Add Python to PATH"**. Ini penting supaya kita bisa menjalankan Python dari command line.
2. Klik **"Install Now"** dan tunggu proses instalasi selesai.
3. Setelah selesai, klik **"Close"**.

## Langkah 3: Cek Instalasi

1. Buka Command Prompt (tekan `Win + R`, ketik `cmd`, lalu tekan Enter).
2. Ketik perintah berikut untuk mengecek versi Python:
   bash
   python --version
   
   Jika muncul versi Python, berarti instalasi berhasil! 🎉

## Langkah 4: Menginstall IDE (Opsional)

Untuk memudahkan coding, kamu bisa menginstall IDE seperti:

- **PyCharm**: [Download PyCharm](https://www.jetbrains.com/pycharm/download/)
- **Visual Studio Code**: [Download VS Code](https://code.visualstudio.com/)

## Langkah 5: Menjalankan Program Pertama

1. Buka editor (misalnya Notepad atau IDE yang sudah diinstall).
2. Ketik kode berikut:
   python
   print("Hello, World!")
   
3. Simpan file dengan ekstensi `.py`, misalnya `hello.py`.
4. Kembali ke Command Prompt, navigasi ke folder tempat file disimpan, lalu jalankan:
   bash
   python hello.py
   

Selamat! Kamu sudah berhasil menjalankan program Python pertamamu! 🎊
