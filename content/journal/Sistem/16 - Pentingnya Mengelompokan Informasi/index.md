saya sering sekali bingung menaruh letak file yang sedang ingin digunakan

kalau filenya hanya sedikit mungkin saya cepat mencarinya tapi kalau filenya banyak ini yang membuat saya pusing

apalagi kalau lupa nama filenya apa, tapi diingatan berkata 'yang kaya gini-nih filenya. namanya apa yah...aduh segala lupa lagi...apa udah kehapus yah ?...'

ini adalah hal yang membuat saya pusing dan harus segera dicarikan solusinya


setelah merenung dalam diam, sayapun menemukan soluisnya yaitu dengan memetakan informasi.

dengan memetakan informasi maka berarti saya harus mengolah informasi tersebut dengan cara mengkategorikannya

katakanlah saya ingin mengolah informasi produk yang saya jual

maka langkah yang saya lakukan adalah saya membuat folder produk

ketika ada produk yang ingin ditambahkan saya akan membuat folder baru nama produknya

misalnya nama produknya 'tiktok downloader' maka saya akan membuat folder seperti nama produknya

semua yang berkaitan degan produk 'tiktok downloader' tersebut akan saya masukan kedalam folder ini

***

contoh lain, memanajemen link url bacaan yang menarik

saya sering sekali membookmark url web tertentu yang menurut saya menarik

karena seringnya saya melakukan hal tersebut url tersebut makin lama makin banyak dan membuat berat browser saya

padahal saya jarang sekali membukanya, tapi jika sewaktu-waktu itu dibutuhkan sulit juga untuk dihapus

maka saya memindahkan list url yang ada ke dalam file txt dan ditumpuk semua menjadi satu

dan ini adalah awal dari kepusingan saya

ketika saya ingin mencarinya lagi saya bingung ada dimana url tersebut...

saya buka satu-persatu filenya ternyata banyak sekali url yang ada dan butuh untuk menyortirnya terlebih dahulu 

maka solusi yang saya lakukan adalah memetakan url tersebut termasuk kategori apa

misalnya url tersebut tentang produk digital, maka saya akan membuat file txt dengan nama 'produk digital.txt' 

semua url yang berkaitan dengan produk digital akan saya masukan kedalam file ini

maka ketika saya mencari referensi yang berkaitan dengan produk digital bisa dengan membuka file ini

saya jadi tidak perlu menyortirnya lagi karena url didalamnya sudah dipastikan hanya berisi web-web produk digital
