aplikasi website ini saya pakai untuk mengatasi keresahan saya dalam menulis sebuah artikel.

saat itu saya ingin sekali bisa menulis secara terstruktur berdasarkan topik, maka terciptalah aplikasi ini

seiring berjalannya waktu aplikasi web ini saya kembangkan sampai ada fitur transaksi didalamnya, misal saya ingin menjual sebuah topik yang saya tulis. maka user harus membelinya terlebih dahulu untuk melihatnya

singkat cerita aplikasi ini sudah menemani saya selama 3 tahun, namun ditahun 2025 ini saya sudah tidak mempertahankannya lagi karena tidak bisa bayar biaya servernya yang terlalu mahal
